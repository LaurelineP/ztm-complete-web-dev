Arrays

Using the following array applies all the bellow actions

 var myArray = ["Banana", "Apples", "Oranges", "Blueberries"];
    1. Remove the "Banana" from the array
    2. Sort the array in order.
    3. Put "Kiwi" at the end of the array.
    4. Remove "Apples" from the array.
    5. Sort the array in reverse order.
      Not in an alphabetical order, but alphabetically reversed:
      Example: the current Array i.e. ['a', 'c', 'b'] becomes ['b', 'c', 'a']

You should have at the end ["Kiwi", "Oranges", "Blueberries"]

Using the following array applies all the bellow actions

 var myArray2 = ["Banana", ["Apples", ["Oranges"], "Blueberries"]];
  • access the "Orange" value
